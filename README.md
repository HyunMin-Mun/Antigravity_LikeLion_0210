# 🚀 WorkFlow MVP

> **"데이터와 AI가 대신 일해주는 전략적 가시성 플랫폼"**

WorkFlow는 복잡한 다중 프로젝트 환경에서 팀의 업무 가시성을 극대화하고, AI가 전략적 우선순위를 제안하는 인텔리전트 업무 관리 도구입니다.

---

## ✨ Key Features

- 🎯 **Strategic Priority Engine**: 영향도와 긴급도를 분석하여 최적의 업무 순위 자동 산출
- 📊 **Hybrid View System**: 칸반 보드와 리스트 뷰 간의 실시간 전환 및 상태 유지
- 🤖 **AI Strategy Assistant**: Gemini 2.0 기반의 업무 리스크 분석 및 인력 재배치 제안
- 🔐 **Enterprise Auth**: Firebase 기반의 보안 인증 및 데모 퀵 로그인 지원
- 💎 **Premium UI/UX**: Indigo & White 테마의 감각적인 인터페이스 및 한글 최적화

---

## 🛠 Tech Stack

- **Core**: React 18, TypeScript, Vite
- **Styling**: TailwindCSS v4
- **Backend**: Firebase (Auth, Firestore)
- **AI**: Gemini 2.0 Flash API

---

## 🚦 Quick Start

### 1. 환경 설정
Firebase 설정 정보(`src/lib/firebase.ts`)를 확인하십시오.

### 2. 로컬 개발 서버 실행
```bash
npm install
npm run dev
```

### 3. 로그인 및 테스트
- `로그인` 화면에서 **'멤버 모드'** 또는 **'매니저 모드'** 버튼을 클릭하여 즉시 기능을 체험할 수 있습니다.
